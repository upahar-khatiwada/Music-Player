### Prerequisites

- Flutter SDK installed
- Android/iOS device or emulator


## Installation & Usage

### 1. Clone the repository.

```bash
git clone https://github.com/upahar-khatiwada/Music-Player
cd Music-Player
```

### 2. Get the dependencies.

```bash 
flutter pub get
```

### 3. Run the code.

```bash
flutter run
```
